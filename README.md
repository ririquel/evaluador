# Evaluador de Agresividad con Machine Learning.



## Para ejecutar 

1. Instalar python y agregarlo al Path

2. Instalar los requisitos con pip:

   ```bash
   pip install -r requieriments.txt
   ```

3. Ejecutarlo con python con el comando:

```
py main.py
```
